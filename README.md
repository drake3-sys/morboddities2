# Morboddities — Vite + React + TypeScript + Tailwind

## Quick start
```bash
npm install
npm run dev
```
Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
1. Push this folder to GitHub
2. Import repo in Vercel → Deploy
3. Add your domain in Vercel and update DNS in GoDaddy
